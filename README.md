# ebec2020
Website for the 2020 European BEST Engineering Competition (EBEC), organized by the Board of European Students of Technology (BEST) Lisbon.
## Tech Stack
- [HTML - HyperText Markup Language](https://html.com/)
- [CSS - Cascading Style Sheets](https://www.w3.org/Style/CSS/Overview.en.html)
- [JS - JavaScript Programming Language](https://www.javascript.com/)
- [Bootstrap - HTML, CSS and JS Library](https://getbootstrap.com/)
- [Node.js - Open-source JavaScript Runtime Environment](https://nodejs.org/en)
- [npm - Package manager for JavaScript](https://www.npmjs.com/package/npm)
